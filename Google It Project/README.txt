***Bing It***

Jason Durkee - #1 & #2
Ty Rozell - #3
Ryan Schick - #4 & #5
Yash Chulki - #6

    CSCE 2110 Project 1. This project reads in a collection of documents, stores and preforms Porter stemming on them, and 
determines which documents in the collection are most similar to user defined query via term frequency-inverse document 
frequency and cosine similarity.

---NO BONUS---

Once unzipped, all the files are in the same directory called "bing-it".
Have all the files being read from in this same directory as the source code files.

Compile Instructions:
To compile, run the following command: g++ *.cpp -o main.out
To run, run the following command: ./main.out
